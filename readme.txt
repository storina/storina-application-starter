storina application starter is woocommerce application with building panel in wordpress admin

== Description ==

Storina Application Starter is plugin provide require webservice for specific android woocommerce application. 
Using woocommerce building panel you can send building request to queue system. after a maximux one day your application is build and you can get apk url.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/storina-application-starter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. go to Application Setting > build panel > Build request and request for request to building application.
4. after maximum one day you can get android apk url in Application setting > build panel > Build details.
